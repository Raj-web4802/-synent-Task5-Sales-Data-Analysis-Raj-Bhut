# Superstore Sales Data Analysis

## Problem Statement
Analyze business performance using the Superstore Sales dataset by exploring revenue patterns, identifying top-selling products, and uncovering sales distribution trends across different categories, regions, and customer segments.

## Dataset
Superstore Sales Dataset (revenue-focused version without profit, quantity, or discount data)

## Tasks Performed
- Loaded the dataset and inspected data structure and shape in VS Code
- Checked for missing data points and handled data integrity 
- Converted order and shipping date columns into datetime format
- Extracted month and year metrics to track historical sales trends over time
- Aggregated total sales figures by product categories and sub-categories
- Analyzed and segmented sales distribution by geographical regions and customer types
- Formatted data observations and noted limitations regarding missing profit columns

## Tools Used
- Python
- Pandas
- NumPy
- VS Code

## Results
Successfully generated comprehensive revenue performance insights and established a structured framework for tracking business sales trends.

## Conclusion
The analysis successfully isolated key revenue drivers across categories and geographic regions, establishing a clear picture of sales distribution while identifying the precise data extensions needed for future profit-margin modeling.
